# Notion Clone Project

## 📚 프로젝트 소개

**Notion Clone Project**는 Vanilla JS를 사용하여 SPA로 구현된 노션 클로닝 프로젝트 입니다.

- 팀명 : TEAM5
- 팀원 : 이정수, 신중석, 윤진영, 하미리
- 기간 : 2024년 12월 16일 (월) ~ 2024년 12월 19일 (목)

## 📍 프로젝트 목적

1. 노션 페이지 에디터와 문서관리 기능을 구현합니다.

- 노션의 핵심기능을 구현합니다.

2. 팀원 전원이 과정에 적극적으로 참여하고 학습합니다.

- 기술적 성장 뿐 아니라 협업을 통한 인간적 성장을 목표로 합니다.

## 💡 주요 기능

### 문서 관리

- 문서의 제목과 내용을 수정할 수 있습니다.
- 문서 수정시 자동저장할 수 있습니다.

### 사이드바

- 최상단 루트 문서를 생성할 수 있습니다.
- 문서 마다 하위 문서를 생성할 수 있습니다.

## ✏️ Code convention

### 커밋 메세지 컨벤션

- 🚨 Fix: [수정 대상] - [수정 내용]
- ✨ Feat: 새로운 기능 추가, 사용자 입장에서 변화가 있을 경우
- 🧹 Chore: 그 외 자잘한 수정에 대한 커밋, 주석, 의존성 설치, 리드미 수정
- 💄 Style: CSS, styled-component 스타일 관련 변경
- 🔨 Refactor: 코드 리팩토링에 대한 커밋, 사용자 입장에서 변화가 없는 코드, 파일명 폴더명 변경 및 이동
- 🗑️ Remove: 파일을 삭제하는 작업만 수행하는 경우
- ♻️ Format: 코드 포맷팅 변경에 관련된 작업

### 코드 컨벤션

- tool : ESLint, Prettier
- [Air-bnb Javascript Style Guide](https://github.com/airbnb/javascript)
